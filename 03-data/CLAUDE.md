# Data Context

Load the specific file matching the data domain in the question.

| File | When to load |
|---|---|
| `glossary.md` | When metric definitions or terminology are needed |
| `engagement-metrics.md` | Engagement metric questions related to product usage |
| `activation-metrics.md` | Activation or onboarding metric questions |
| `locations.md` | Location level questions |
| `date-conventions.md` | Any query with date filtering, grouping by period, period-over-period comparisons, or cohort day windows (D1, D14, D30, etc.) |
| `product-domains/timetracking.md` | Timecards, clock-in/out, breaks, manager edits, ACO/ACI |
| `product-domains/scheduling.md` | Shifts, schedules, open shifts, publishing, shift edits, shift trades |
